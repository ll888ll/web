# Archivos archivados (legacy)

Se movieron a esta carpeta para mantener la raíz limpia. Ninguno de estos archivos participa en los workflows actuales ni en la entrega de DNS/telemetría. Pueden restaurarse con `git mv trash/docs_legacy/<archivo> ./`.

| Archivo                                | Motivo                                                                 |
| -------------------------------------- | ---------------------------------------------------------------------- |
| `INSTRUCCIONES.md`                     | Manual antiguo del sensor CICF (no forma parte del entregable actual). |
| `INTEGRACION.md` / `integracion*.md`   | Notas previas de integración Croody/APIs reemplazadas por docs vigentes. |
| `funcionalidades.md` / `mejoras_propuestas.md` | Ideación temprana ya consolidada en `manual_tecnico.md`.          |
| `seguridad_y_escalabilidad.md`         | Documento exploratorio, no citado en los informes finales.              |
| `teoria_aplicada.md`                   | Referencias conceptuales no requeridas en la entrega.                   |
| `uso_y_mantenimiento.md`               | Manual antiguo de otra versión del sistema.                             |
| `informe_entrega2_telemetria.md`       | Entrega previa (conservar aquí solo como histórico).                    |

Si algún documento vuelve a ser necesario, muévelo fuera de `trash/` y actualiza las referencias correspondientes.
